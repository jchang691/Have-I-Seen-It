== Welcome to HaveISeenIt

This site keeps track of your movies library as well as if you have seen it or not.

== Let's Get Started

1. Create an account from main page if you have not signed in yet.
2. Next, add a movie by either browsing the "All Movies" tab, or adding a 
   new movie by clicking the "Add Movie" button
3. Next to each movie, there's a link "Seen" which allows you to keep track of which movies you have seen
4. Enjoy!
